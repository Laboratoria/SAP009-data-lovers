Quem nós somos?

Olá, esse é o nosso segundo Projeto no Bootcamp da Lab! Estamos muito felizes com o resultado do Projeto, esperamos que vocês também gostem e aproveitem para conhecer um pouco mais os personagens do Studio Ghibli.

Neste projeto tinhamos como objetivo construir uma página web para visualizar um conjunto (set) de dados que se adequasse às necessidades do usuário, sendo elas, visualizar dados, filtrá-los, ordená-los e obter médias(ou outros valores) através de um banco de dados sobre determinado assunto.
